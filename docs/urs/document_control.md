# URS — Document Control

| Section | Content |
|--------|---------|
| **Purpose** | Controlled SOPs/WIs with versioning, approval, and periodic review. |
| **Users** | Document owners, QA, all readers. |
| **Functional requirements** | Draft/review/approve; effective/supersede; retraining trigger; review log. |
| **Regulatory** | 21 CFR Part 11, EU GMP 4.1, ALCOA+. |
| **Acceptance** | Only one effective version; signatures on approval; review reminders. |
