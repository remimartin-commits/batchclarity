# URS — Manufacturing Execution (MES)

| Section | Content |
|--------|---------|
| **Purpose** | Execute batch records, enforce MBR, release with controls. |
| **Users** | Operators, line supervisors, QA for release. |
| **Functional requirements** | MBR version control; step execution; contemporaneous sign-off; no back-fill. |
| **Regulatory** | EU GMP Annexe 1, 21 CFR 211, ALCOA+ data integrity. |
| **Acceptance** | EBR with electronic signatures on critical steps and batch release. |
