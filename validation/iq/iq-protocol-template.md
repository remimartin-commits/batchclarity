# Installation Qualification (IQ) — Protocol template

1. **Scope** — System name, version, environment (test/validation), location.
2. **Prerequisites** — Hardware, OS, network, access accounts.
3. **Installation steps** — Reference runbook; record commands and outputs.
4. **Configuration capture** — Config files, feature flags, DB URL (sanitized), TLS.
5. **Security baseline** — Firewall rules, service accounts, secret storage.
6. **Evidence** — Hashes, screenshots, logs attached in validation package.
7. **Deviations** — Log, assess, and resolve before sign-off.
8. **Sign-off** — IT/QA/owner signatures with dates (electronic or wet).
